# _ai-context — AI Session Context Archive

This folder contains structured knowledge files for AI assistants working on the Astra RPG Health documentation. Read all four files at the start of every new session.

## Files

| File | Read when... |
|---|---|
| [`project-overview.md`](project-overview.md) | Always — package architecture, key classes, business rules, folder paths |
| [`style-guide.md`](style-guide.md) | Before writing or editing any documentation text |
| [`documentation-progress.md`](documentation-progress.md) | To understand what is done and what remains; **update this file** when completing a section |
| [`code-reference.md`](code-reference.md) | Before opening source files — maps doc sections to their source files |

## How to Use (for AI assistants)

1. At the start of every session, read all four files above.
2. Check `documentation-progress.md` for the current status and next steps.
3. Before writing a section, consult `code-reference.md` to identify which source files to read.
4. Follow `style-guide.md` strictly — tone, formatting, callout boxes, image conventions.
5. After completing a section, update `documentation-progress.md`.
