Class EntityAttributes

Namespace
SoapRpgFramework.Attributes
Assembly
com.electricdrill.soap-rpg-framework.Runtime.dll

Manages the attributes of an entity. This component calculates attribute values by combining base values, flat modifiers, percentage modifiers, and spent attribute points. It requires an SoapRpgFramework.Attributes.EntityAttributes.EntityCore component on the same GameObject.

[DisallowMultipleComponent]
[RequireComponent(typeof(EntityCore))]
public class EntityAttributes : MonoBehaviour
